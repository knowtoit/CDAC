# Question 1

Contains three programs:

1. `PrimeCommandLine` accepts 3 command-line numbers, prints table for prime numbers, otherwise divides by 10.
2. `StudentDetailsTest` accepts details for 2 students and displays them.
3. `StudentMenuTest` is a menu-driven program for multiple students with search and GPA calculation.

Compile from this folder:

```powershell
javac -d out src\com\assignment\q1\*.java
```
